## 目录约定

- `concept/`：角色设定图、线稿、主视觉方向。
- `turnaround/`：三视图、上色三视图、建模/像素化参考。
- `animation/`：帧动画设定、动作拆分、sprite sheet 前置参考。
- `references/`：风格、文明符号、武器、材质、同类角色参考。
- `notes/`：后续评审、需求变更、制作拆分。
- `_source/`：若未来需要保留云文档原始素材包，可放这里。
