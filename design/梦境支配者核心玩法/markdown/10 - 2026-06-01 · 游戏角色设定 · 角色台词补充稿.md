## 6.1 战士猫：塞班 / 圣剑士

| 触发场景 | 台词 | 表演提示 |
|-|-|-|
| 登场 | 以银誓之名，床前一寸，不许越界。 | 克制、端正，像宣读骑士誓约。 |
| 待机 | 盾在，梦就在。小鱼干稍后再说。 | 前半句庄严，后半句小声心虚。 |
| 普攻 | 圣剑，斩噩梦。 | 短促、有力。 |
| 小技能一：圆盾冲锋 | 让开，别碰主人的枕头！ | 瞬间破功，护主感拉满。 |
| 小技能二：银誓护盾 | 敌意冲我来。 | 低沉、可靠。 |
| 大招：王冠圣裁 | 王冠之下，梦魇退散！ | 庄严爆发，尾音抬起。 |
| 受击 | 盔甲无碍，胡须还在。 | 假装冷静，带一点猫式自尊。 |
| 残血 | 骑士不会后退，最多先舔一下爪。 | 强撑但可爱。 |
| 护床 | 越过这条线，就是宣战。 | 压低声音，警告感。 |
| 切换退场 | 防线交给你，别让它们靠近。 | 信任队友，语速稳。 |
| 胜利 | 誓约完成。现在可以领零食了吗？ | 先正式，后半句露出本性。 |
