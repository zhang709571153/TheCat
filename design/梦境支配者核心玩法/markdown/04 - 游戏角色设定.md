### 6.1 战士猫：塞班/ 圣剑士

| **权柄 / 属性** | 誓约 / 太阳 |
|-|-|
| **机制落点** | 圣剑、护盾与王冠圣裁落在圣光、裁决、守护辉光方向，不按火球处理。 |

| 品种 | 英国短毛猫，银渐层 |
|-|-|
| 参考文明 | 凯尔特文明 + 中世纪不列颠骑士文化 |
| 性格 | 冷静、克制、守礼，有骑士誓约感。 |
| 反差 | 外表像小小骑士王，看起来很严肃，但本质是圆脸短腿银渐层，看到小鱼干和猫马上放下身段。很喜欢主人（其实是主人毒唯），如果最后猫咪统治世界了也还是愿意继续饲养主人的，不高兴的脸内心是：喵喵喵喵喵喵！零食零食零食 |
| 武器 | 银月圣剑 + 小圆盾 |
| 定位 | 前排、防守、击退、护床。 |

| 类型 | 技能名 | 技能类型 | 说明 |
|-|-|-|-|
| 被动 | 银誓壁垒 | 守床联动 | 塞班获得护盾时，主人床也获得一层弱护盾。 |
| 被动 | 炽誓锋芒 | 盾转输出 | 塞班有护盾时，普攻和技能命中会额外追加一段太阳 / 火元素伤害。 |
| 普攻 | 圣剑连斩 | 近战攻击 | 近距离三段挥剑。若塞班拥有护盾，可通过“炽誓锋芒”追加太阳 / 火元素伤害。 |
| 小技能一 | 誓约护盾 | 护盾 | 塞班生成中量护盾，是开盾站前排和触发“炽誓锋芒”的核心技能。 |
| 小技能二 | 圣剑横扫 | 范围伤害 / 击退 | 向前挥出太阳剑弧，造成范围伤害并击退敌人，用于清理靠近床的怪群。 |
| 小技能三 | 誓约战旗 | 阵地增益 | 在地面插下一面誓约旗帜，范围内友军造成的伤害提高。 |
| 小技能四 | 炽阳冲锋 | 位移 / 击退 / 补盾 | 塞班向前冲刺，撞开路径敌人，并获得少量护盾。 |
| 大招一 | 圣盾裁返 | 防守反击 | 塞班获得大量护盾，并在持续期间阻挡飞行道具；技能结束时，对范围内敌人造成等同于护盾减少量的太阳伤害。护盾减少量只计算被敌人打掉和阻挡飞行道具消耗的部分。 |
| 大招二 | 王冠圣裁 | 爆发伤害 | 召唤太阳圣剑砸向目标区域，造成大范围火元素伤害；塞班当前护盾越高，伤害越高。 |

### 6.2 法师猫：奈芙蒂斯 / 月沙特工

| **权柄 / 属性** | 支配 / 大地 |
|-|-|
| **机制落点** | 方尖碑、流沙、金字塔、沙漠龙卷与王庭沙尘暴都指向大地和支配，强调区域压制、减速 / 禁锢、召唤物统御与反飞行单位。 |

| 品种 | 阿比西亚猫，金黄色 |
|-|-|
| 参考文明 | 古埃及，新王国时期 |
| **名字来源** | “奈芙蒂斯”取自古埃及神话中的 Nephthys（涅芙蒂斯 / 奈芙蒂斯），九柱神之一，常与夜晚、墓葬、边界和守护死者相关。本角色借用其古埃及神秘女性权威与陵墓守护气质，但不直接等同原神话角色。 |
| 性格 | 傲娇、自信、优雅，有点看不起敌人。 |
| 反差 | 嘴上说“这种低级梦魇不值得本猫出手”，但一旦床被碰到会立刻认真。 |
| 武器 | 漂浮在手边的象形文字金字塔魔方。 |
| 定位 | 范围输出、清怪、区域压制。 |

| **类型** | **技能名** | **机制说明** |
|-|-|-|
| 被动 | **沙缚猎令** | 奈芙蒂斯及其召唤物对减速或禁锢目标造成的伤害提高。 |
| 被动 | **王庭增筑** | 奈芙蒂斯的召唤物存在上限 +1。 |
| 普攻 | **符文弹** | 发射小型象形文字能量弹攻击敌人，对减速或禁锢目标可以吃到“沙缚猎令”的增伤。 |
| 小技能一 | **月沙方尖碑** | 召唤一座方尖碑，自动攻击范围内敌人并造成减速。基础最多同时存在 2 座；拥有“王庭增筑”后最多 3 座。 |
| 小技能二 | **流沙陷阱** | 在地面生成流沙区域，范围内敌人被减速；停留过久会被短暂禁锢。 |
| 小技能三 | **镇沙金字塔** | 召唤小型金字塔阻挡敌人；存在期间周期性释放沙石冲击波，造成伤害和减速；被打坏时爆炸，对范围内敌人造成伤害。 |
| 小技能四 | **王碑飞弹** | 从方尖碑前方发射沙金飞弹，命中或到达最远距离后爆炸，造成范围伤害并禁锢敌人；场上没有方尖碑时，从奈芙蒂斯面前发射。 |
| 大招一 | **王庭沙尘暴** | 召唤覆盖全屏的沙尘暴，造成全屏减速和大量沙尘伤害；所有飞行单位被击落，落地后短暂禁锢。 |
| 大招二 | **沙海龙卷** | 在指定圆形范围内掀起沙漠龙卷风，持续一段时间，对范围内敌人造成持续沙尘伤害。 |

### 6.3 牧师猫：铃音 / 安眠巫女

| **权柄 / 属性** | 音律 / 月亮 |
|-|-|
| **机制落点** | 铃音以持续技能、治疗触发增伤、冰属性减速和场地回复为核心。月亮属性主要表现为冰符、冰月鸟居、冷光治疗与冰附魔；音律权柄表现为铃声持续留场、治疗节奏和旋舞范围效果。 |

| 品种 | JP短尾猫，三花或白底花色 |
|-|-|
| 参考文明 | 日本平安时代 + 神道巫女形象 |
| 性格 | 嘴硬、护短、急性子，猫本能很重：不高兴会压耳朵、甩短尾、哈气，治疗时一边把队友拍回安全区，一边骂他们乱跑。她不是温柔哄人的治愈系，而是凶巴巴把你按住治疗的猫咪巫女。 |
| 反差 | 职业是治愈和安眠巫女，技能看起来像铃声、符纸、结界这种柔和系，实际本人领地意识很强、骂骂咧咧、动不动炸毛。嘴上嫌“麻烦死了”，身体却会第一时间补血、护床、稳住全队。 |
| 武器 | 神月铃、冰符、治愈铃铛、冰月鸟居。 |
| 定位 | 持续治疗、冰属性范围伤害、减速控场、治疗后增伤、饱腹度恢复。 |

| **类型** | **技能名** | **技能类型** | **说明** |
|-|-|-|-|
| 被动一 | **余音绵延** | 持续强化 | 铃音所有持续类技能持续时间延长 20%。 |
| 被动二 | **愈音鼓舞** | 治疗增伤 | 任意角色受到铃音的治疗效果时，短时间内获得伤害提升。 |
| 普攻 | **月铃音波** | 中距离攻击 | 挥动神月铃，发出中距离冰月音波，对小范围敌人造成冰属性伤害。 |
| 小技能一 | **神月铃击** | 三段伤害 / 治疗 | 向前挥舞神月铃，对前方范围敌人造成冰伤和减速，同时治疗周围友军；此技能可连续释放 3 次。 |
| 小技能二 | **治愈铃铛** | 持续治疗 / 资源回复 | 召唤一个铃铛，持续治疗场上角色，并恢复一定饱腹度。 |
| 小技能三 | **冰符环护** | 护身持续伤害 | 召唤冰符环绕自己，冰符会周期性伤害周边敌人。 |
| 小技能四 | **霜月旋舞** | 持续范围伤害 / 减速 | 铃音持续旋转一段时间，对范围内敌人造成冰属性伤害并减速。 |
| 大招一 | **冰月鸟居** | 场地持续 / 治疗 | 召唤鸟居，鸟居会周期性对范围内敌人造成冰属性伤害和减速效果，并回复场上角色生命值。 |
| 大招二 | **月铃祈愈** | 全队急救 / 冰附魔 | 恢复所有角色大量生命值；铃音获得一段时间冰附魔，普攻或技能造成额外冰属性伤害。 |

### 6.4 刺客猫：影丸/逐影者（4级解锁）

| **权柄 / 属性** | 战争 / 天空 |
|-|-|
| **机制落点** | 高机动、突进、追击和残影按风属性表达，强调空隙切入与战斗节奏压迫。 |

| 品种 | 暹罗猫，重点色（象牙白身体+深褐色面罩/耳/四肢） |
|-|-|
| 参考文明 | 泰国，古暹罗王朝 + 东南亚密林猎手 |
| 性格 | 好奇、敏锐、闲不住。永远在追逐什么——光斑、影子、或者下一个目标。看似漫不经心，实则全场动态尽收眼底 |
| 反差 | 嘴上说"我只是觉得追着跑比较好玩"，但每次追到的都恰好是最关键的那只觉醒体。看起来在玩，其实是全队最高效的清场者 |
| 武器 | 双持月牙形泰式短弯刀，刀身轻薄如蝉翼，高速挥舞时在空气中留下金色弧光残影。不用时弯刀悬浮在身侧，像两片追随主人的月光 |
| 定位 | 单体爆发、追击、高机动 |

|-|-|-|-|

| 普攻 | 弧光斩 | 近战攻击 | 双弯刀高速交叉斩击，攻速全角色最快。每次命中在敌人身上留下2秒"弧光印记"。 |

| 大招 | 百影追猎 | 多段爆发 | 3秒内极速穿梭，自动锁定并攻击场上所有带"弧光印记"的敌人各一次，每次从背后切入造成2倍伤害。先标记越多，大招总伤越高。冷却18秒。 |

| 小技能一 | 水袖牵云 | 控制 / 牵引 | 甩出水袖拉扯敌人，小型敌人被拉近，大型敌人短暂停顿。若目标带“戏印”，牵引结束时额外造成一次亮相伤害。 |
| 小技能二 | 变相一折 | 控场 / 迷惑 | 展开脸谱扇光，使范围敌人短暂转向或攻击错误目标。该技能不遮住猫脸，脸谱光纹只出现在扇面和周围光效中。 |
| 大招 | 梨园压轴 | 群体爆发 / 控场 | 戏台灯影落下，绛台连续亮相、甩袖、转扇，对带“戏印”的敌人造成多段太阳属性伤害并短暂定身。场上戏印越多，总控场收益越高。 |
