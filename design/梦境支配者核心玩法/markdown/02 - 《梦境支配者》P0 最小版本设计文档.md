## 2. 背景故事

很久以前，猫咪并不是人类的宠物，而是人类文明背后的真正统治者。在猫族自己的史书里，它们曾是十二生肖的第一位，是东方历法的梦境守门者；是尼罗河神庙中被供奉的猫神，是古埃及家庭、月夜与丰饶的主人；是两河流域王权壁画里的狮形影子，是战争、王座与女神身边的古老猫科权柄；也是地中海神话中月亮、夜行、狩猎与神谕背后的秘密管理者。人类以为这些只是神话和民俗，猫族却把它们称为“旧王朝档案”。

那时的猫族拥有一种神性优势：它们不用拉屎，也不会被吃饭、睡觉和卫生问题打断，因此能连续潜入人类梦境，用呼噜声发布神谕，指挥人类耕种、祭祀、建造神庙，并把最舒服的位置献给猫。

后来，亚瑟王进入梦域，击败了旧王朝最强的“猫猫大魔王”。他没有向圣杯许愿永生，而是许下了一个更阴险的愿望：让所有猫猫都变得可爱，并且从此需要吃东西、睡觉、拉屎。

圣杯诅咒生效后，人类再也无法把猫当成严肃的统治者，只会觉得它们可爱。猫族被迫从王座退到沙发，从神坛退到窗台。而那些曾经属于猫族的神权，也在旧王朝崩塌时四散遗落：有的沉入尼罗河的神庙壁画，有的埋进两河流域的王权泥板，有的化作月夜祭坛上的残响，有的被藏进人类口耳相传的神话里。

直到现代，猫猫们发现了一条重返失落之地的道路：人类的梦境。梦会记住文明忘记的东西，也会打开通往神话残骸的门。只要潜入人类梦中，猫猫们就能抵达那些被遗忘的古老国度，找回属于自己的神权。

于是，猫族的千年大计，重新开始。

### 七执政：旧日神权的七个席位

在旧王朝档案中，猫族并不把神权称作“技能”或“元素”，而称作七执政。七执政不是七位单独的神明，而是七个曾经支撑猫族统治世界的神权席位。每一个席位都代表一种统治梦境与文明的根本能力：有的让梦保持节律，有的让誓言成为边界，有的让万物听从命令，有的让未来提前显影，有的让生命循环不息，有的让战争服从猫爪的意志。

圣杯诅咒降下后，七执政随旧王朝一同崩塌。音律、誓约、支配、预言、丰收、战争等权柄被撕成碎片，散落在不同文明的神话残骸之中。而第七席的名字，则在猫族史书里被整页抹去，只留下一个空白的王座。猫猫们重新进入人类梦境，不只是为了取回力量，也是为了找出第七执政为何失落，以及旧王朝真正灭亡的原因。

### 3.1 玩法定位

P0 核心玩法是一局 10 层的随机路线守床肉鸽。玩家从 1 只初始小猫开始，每层先在路线地图上选择节点，再进入战斗、事件、商店、伙伴或祝福供奉。最终目标是在床铺没有被破坏前抵达第 10 层并击败 Boss。

### 3.2 P0 关键规则

| 模块 | 主规则 |
|-|-|
| 路线 | 第 1 层固定守床，第 10 层固定 Boss；第 2 到第 9 层随机出现 2 到 4 个可达节点。 |
| 伙伴 | 开局第一只猫和后续伙伴第一次入队时，都会立刻触发一次该猫专属小技能四选一；未入队小猫不会进入可升级名单。特定层会出现伙伴机会。 |
| 技能池 | 每只小猫拥有 2 个被动、4 个小技能、2 个大招。战斗获得全队共享经验，经验满后选择 1 只已入队小猫升级；升级阶段依次为被动二选一、小技能四选一、大招二选一。 |
| 大招 | 某只猫开局 / 入队先学 1 个小技能，完成第 1 次和第 2 次升级后拥有 1 个被动 + 2 个小技能；第 3 次升级时从该猫 2 个大招中二选一。 |
| 祝福 | 祝福不是每层发放，主要来自祝福供奉、商店小概率刷新或精英小概率掉落；祝福只承载战斗类构筑，分权柄、属性、数值和机制方向。 |
| 事件道具 | 原通用构筑效果不再作为祝福，改为事件道具；P0 只保留 5 个，默认通过梦境事件获得。 |
| 小鱼干 | 战斗后获得小鱼干，可在商店购买补给、刷新货架、升级已有祝福或小概率购买新祝福。 |
| 技能升级 | 商店技能升级只增加数值，例如伤害、护盾量、治疗量、持续时间、冷却时间、范围、召唤物生命或持续时间；不改变技能机制。 |

**修整猫窝：**安全恢复节点，进入后无怪物，地图包含床、猫砂盆和喂食器；三处各可互动一次，分别用于清空屎意值、恢复饱食度、修复床一定比例生命。

## 4. 战斗系统

战斗采用轻割草 + 中心床防守。猫咪普通攻击可以自动索敌，玩家主要负责走位、切换猫咪、释放技能和处理猫咪本能需求。

### 4.1 权柄与属性

<callout emoji="📌">
**当前口径：**角色识别采用「权柄 + 属性」二维体系。属性负责美术表现和通用战斗表现；权柄只负责战斗机制关键词。权柄不写美术表现，每个权柄只保留一个战斗关键词。两者不做固定绑定，可自由组合。
</callout>

#### 属性体系：美术表现与战斗表现

| 属性 | 美术表现关键词 | 战斗表现关键词 | 通用战斗表现 |
|-|-|-|-|
| 大地 | 沙石、尘环 | 护盾、坚守 | 大地属性角色释放大招后，给自己获得一层护盾。 |
| 天空 | 绿风、风刃 | 击飞、风压 | 敌人在短时间内被天空属性技能连续命中，会被击飞。 |
| 海洋 | 潮纹、水光 | 回复、回流 | 海洋属性角色释放大招后，回复自己和附近友军少量生命。 |
| 太阳 | 赤焰、火光 | 灼烧、爆燃 | 敌人在短时间内被太阳属性技能连续命中，会进入灼烧状态。 |
| 月亮 | 白冰、冷光 | 减速、冻结 | 敌人在短时间内被月亮属性技能连续命中，会被大幅减速；叠满后短暂冻结。 |

#### 权柄体系：单关键词战斗表现

| 权柄 | 战斗关键词 | 战斗方向 |
|-|-|-|
| 音律 | 持续 | 延长持续技能、持续区域、持续效果。 |
| 誓约 | 护盾 | 生成护盾、强化护盾、围绕护盾触发效果。 |
| 支配 | 召唤 | 召唤物、炮台、侍从、指令单位。 |
| 预言 | 冷却缩减 | 缩短技能冷却，让角色更频繁释放技能。 |
| 丰收 | 回复 | 治疗、回血、续航、生命恢复。 |
| 战争 | 普攻 | 普攻强化、连击、追击、普攻特效。 |

设计时先看权柄，再套属性：权柄决定技能机制骨架，属性决定技能视觉材质和元素反馈。例如「誓约 + 太阳」可以是火焰护盾，「支配 + 大地」可以是沙石召唤物，「音律 + 月亮」可以是冰霜持续音域。

| 系统 | 说明 |
|-|-|
| 移动 | 控制当前猫咪在卧室梦域中移动，围绕床建立防线。 |
| 普通攻击 | 自动索敌，降低操作压力，让玩家专注站位和切换。 |
| 小技能 | 每只猫一个核心技能，承担职业职责。 |
| 大招 | 清场、防守或救场技能，作为局势逆转手段。 |
| 三猫切换 | 战士挡线，法师清怪，牧师恢复主人睡眠度。 |
| 猫咪本能 | 猫咪会饿、会想拉屎，玩家需要切换或处理场景道具。 |

### 5.1 核心数值概览

P0 核心数值由四项构成：主人睡眠度、猫咪血量、队伍屎意值、队伍饱肚度。它们分别制造守床压力、生存压力、节奏打断和补给循环。

| 数值 | 定位 | 玩家压力 | 主 PO 保留结论 |
|-|-|-|-|
| 主人睡眠度 | 本局失败条件 | 床被攻击或梦境被干扰时下降 | 守床是最核心目标，睡眠度归零即失败。 |
| 猫咪血量 | 单只猫生存状态 | 低血量迫使玩家切换角色或寻找恢复 | 切猫不重置血量，只改变当前承压对象。 |
| 队伍屎意值 | 队伍共享节奏压力 | 满值会触发原地拉屎，降低睡眠度上限 | 不能靠切猫规避，主要通过猫砂盆处理。 |
| 队伍饱肚度 | 队伍共享补给压力 | 随战斗、技能和特殊怪物影响下降 | 喂食器恢复饱肚度，但会带来消化后的屎意增长。 |

### 5.2 单局节奏

基础循环是：战斗清怪 → 饱肚度下降 → 使用喂食器 → 进入消化状态 → 屎意值加速上升 → 使用猫砂盆 → 回到防守。这个循环让玩家在守床、换猫、补给和处理猫咪本能之间不断做取舍。

### 6.1 初始三猫

- **塞班 / 圣剑士：**前排、防守、击退、护床，负责在敌潮压近时重建防线。
- **奈芙蒂斯 / 月沙特工：**范围输出、区域压制、减速与禁锢，负责清怪和控场。  
**名字来源：**取自古埃及神话中的 Nephthys（涅芙蒂斯 / 奈芙蒂斯），其形象常与夜晚、墓葬、边界和守护死者相关；角色仅借用其古埃及神秘女性权威与陵墓守护气质。
- **铃音 / 安眠巫女：**治疗、护盾、驱散、救场，负责维持队伍和主人睡眠状态。

### 6.2 升级解锁角色

- **影丸 / 逐影者：**单体爆发、追击、高机动，适合在敌人被标记或控住后切入收割。
- **棉花 / 梦织女：**标记增伤、减速铺场、切换增益，负责先手布置战场路线和输出窗口。
- **玉衡 / 阴阳令：**反伤、蓄力、Boss 反制，适合处理高频攻击的精英觉醒体和 Boss。

## 7. P0 敌人与 Boss

<callout emoji="💡">
**最新版结论：**P0 敌人从“起床干扰”统一收束为黑泥梦境污染体系。基础怪负责压床，梦轨小火车负责直线冲锋并被护盾单位反制，未读红点小飞虫负责飞行绕床压力，中距离怪负责压场，冷光灯影用坠弹和连续飞弹压低玩家血量并制造走位，坠梦玩具熊作为精英怪负责大范围跳砸爆发，Boss 负责召唤小怪与投掷 APP，整体服务“守床 + 玩家走位 + 清怪优先级 + 角色机制反制”的教学目标。
</callout>

### 7.1 敌人设计总表（最新版）

| 冲锋小怪 | **梦轨小火车** | 直线冲锋 / 护盾反制 | 纯玩具火车形象。原地蓄力后，地面出现直线玩具轨道预警，随后沿轨道向前冲刺；撞到护盾单位时，小火车会被短暂眩晕，眩晕时间固定，和护盾厚度无关。 |
| 中距离怪 | **红眼闹铃** | 范围压场 / 站位干扰 | 缓慢靠近床，在中距离释放周边范围伤害，迫使玩家离开固定输出点。 |

| 精英怪 | **坠梦玩具熊** | 跳跃砸地 / 大范围爆发 | 黑泥污染的毛绒玩具熊。蓄力一段时间后向前跳跃，落地用双掌或玩具重锤砸击地面，对大范围造成伤害。 |

### 7.2 美术统一规则

- **材质统一：**怪物都带有黑色淤泥、梦境裂缝或污染拖痕，和卧室梦境的黑泥污染源保持一致。
- **眼睛统一：**危险识别点使用红色眼睛或红色来电图标，保证战斗中一眼能读出敌意。
- **轮廓区分：**黑泥梦魇是小黑史莱姆，梦轨小火车是玩具火车轮廓，红眼闹铃是闹钟轮廓，冷光灯影是带弹簧机械臂的设计室台灯，未读红点小飞虫是高饱和红点通知轮廓，带白色未读数字、透明小翅膀和黑泥拖尾，坠梦玩具熊是毛绒玩具熊，来电暴君是巨型手机 Boss。
- **攻击读图：**远程和范围攻击必须有清晰预警。梦轨小火车使用直线轨道预警，冷光灯影使用地面预警圆和连续飞弹喷射动作，未读红点小飞虫使用红点集群环绕轨迹、附着床沿和未读数字膨胀，红眼闹铃使用周边震荡圈，坠梦玩具熊使用跳砸落点大圆，Boss 使用 APP 投掷轨迹和召唤裂缝。

### 7.3 单体设计细节

#### 黑泥梦魇

| 外观 | 小小的黑色史莱姆形象，不长手脚。身体像一团软黑泥，只有红色小眼和拖在地上的污泥尾迹。 |
|-|-|
| 行为 | 沿路线缓慢向床靠近，接触床后扣主人睡眠度。适合成群出现，作为最基础的清怪目标。 |
| 作用 | 让玩家理解“不能只打 Boss，也要守床和清线”。 |

#### 梦轨小火车

| 外观 | 纯玩具火车方向，不混入其他角色设定。车体像儿童房里的发条小火车，车头灯被黑泥污染成红色眼睛，轮子和车厢边缘带黑泥拖痕。 |
|-|-|
| 行为 | 进入攻击距离后原地蓄力，车头冒黑泥蒸汽，地面出现一条直线玩具轨道预警。蓄力结束后沿轨道高速冲刺。 |
| 攻击 | 冲刺命中普通角色时造成伤害与击退；冲到床边时会撞击床并扣主人睡眠度。若冲刺撞到护盾单位，小火车会被短暂眩晕，眩晕时间固定，和护盾厚度无关。冲刺结束或被眩晕后都会短暂停顿，给玩家反打窗口。 |
| 作用 | 作为直线冲锋小怪，训练玩家识别长条预警并横向走位。它和红眼闹铃、坠梦玩具熊区分开：火车管直线，闹铃管周身，玩具熊管落点大圆。 |

#### 红眼闹铃

| 外观 | 黑色淤泥闹钟，整体是圆滚软糯的小闹钟玩具感，不再走邪恶裂脸方向。钟面保留红色眼睛作为敌意识别，但表情更像困困的小怪；双铃像软软的耳朵，黑泥滴落只做污染感，不做尖角或恐怖轮廓。 |
|-|-|
| 行为 | 靠近玩家或床到中距离后，在自身周边释放范围伤害。范围伤害前应有震荡或闹铃预警。 |
| 作用 | 打断玩家站桩，让玩家开始学习走位和拉开距离。 |

#### 冷光灯影

| 外观 | 设计室工作用台灯，带弹簧、可拉伸机械臂、关节和拉杆细节。黑泥缠绕在底座和关节上，灯头内有红色眼睛，整体比普通台灯更现代、更工具化。 |
|-|-|
| 行为 | 主要攻击玩家，并缓慢向床靠近。范围内没有玩家时，才转向攻击床。 |
| 攻击 | **技能一：污泥坠弹。**冷光灯影向天上发射一颗污泥飞弹，地面出现圆形范围指示，短暂延迟后飞弹从天而降并造成范围伤害。  <br/>**技能二：连续喷射。**灯头连续抖动并向目标方向喷射多颗小型污泥飞弹，连续造成伤害，用于压低玩家血量。不要做成细线或激光发射。 |
| 作用 | 优先压低玩家血量，让玩家不能一直舒服地围床输出。 |

#### 未读红点小飞虫

**概念：**未读红点小飞虫是人类梦境中“未读消息焦虑”的具象化。它们成群飞舞，像永远清不完的通知一样绕着床，最终贴在睡梦上，把主人从安眠里一点点吵醒。

- **外观：**主体是高饱和红色通知圆点，身体上有白色未读数字，例如 1、9、99+。背后长着透明小翅膀，边缘带黑色淤泥拖尾，表情烦人但轻小，整体读图要像一群会飞的手机红点。
- **行为：**通常 3 到 6 只成群出现，围绕床做弧形或环形飞行，并逐步向床贴近。它们是飞行单位，不受金字塔、防线、流沙等地面阻挡影响。
- **攻击：**接近床后会附着在床边形成未读红点标记，持续消耗睡眠度。玩家靠近时，红点会短促爆鸣，造成小额伤害和短暂打断。
- **反制：**范围伤害和对空控制可以快速清理红点群。塞班的护盾可以保护床沿，奈芙蒂斯的沙尘暴可以击落飞行单位，铃音的范围冰伤适合清群。
- **P0 简化：**第一版只保留三个表现：绕床飞行、附着床边持续扣睡眠度、死亡时爆成红色通知粒子。

#### 坠梦玩具熊

| 外观 | 黑泥污染的毛绒玩具熊，身体胖重，缝线和棉花外露，红色眼睛藏在毛绒阴影里。整体要像儿童房玩具变成梦境怪物，不做真实野兽方向。 |
|-|-|
| 行为 | 发现目标后停下蓄力，身体下蹲、弹簧感压缩，地面出现大圆形落点预警。蓄力结束后向前跳跃。 |
| 攻击 | 落地时用双掌或玩具重锤砸击地面，造成大范围伤害。攻击范围比红眼闹铃更大，但前摇更明显，玩家可以通过提前离开落点规避。 |
| 作用 | 作为精英怪承担爆发威胁，考验玩家读预警、撤离落点和处理攻击后的反打窗口。它是“圆形落点威胁”，与梦轨小火车的“直线轨道威胁”形成区分。 |

#### 来电暴君

| 外观 | 巨大的黑泥手机，从黑色梦境裂缝中探出半个身体。手机带领带，有“老板打电话催你”的压迫感，红色来电图标像眼睛。 |
|-|-|
| 行为 | 作为 P0 最终 Boss，负责召唤小怪并向玩家投掷 APP。APP 可以做成不同图标，但不要出现真实品牌或可读文字。 |
| 作用 | 把 P0 前面学过的清怪、守床、走位、处理远程压力整合到一场 Boss 战里。 |

## 12.关于角色记忆点的讨论

为了让小猫角色登场时更有记忆点，后续角色不再只依赖“职业 + 技能”来区分，而是增加一层更有文化厚度的设定：**圣遗物**。

这里先明确：之前讨论过的“梦令”设定暂时删除，不再作为角色能力体系。新的方向是：圣遗物尽量参考现实古文明中真实存在的文物，游戏中解释为这些文物在梦域中的“猫族旧王朝力量投影”。现实世界里它们是博物馆文物；梦境世界里，它们会显露猫族曾经统治人类时留下的权柄。

### 设定核心

| 设定点 | 说明 |
|-|-|
| 圣遗物来源 | 优先参考真实存在的古文明文物，例如罗塞塔石碑、乌尔军旗、后母戊鼎、曾侯乙编钟、三星堆青铜面具等。 |
| 游戏解释 | 这些文物并不是普通人类遗产，而是猫族旧王朝把力量藏进人类文明后的痕迹。 |
| 角色连接 | 每只重要小猫可以绑定一个圣遗物，玩家先听说圣遗物，再期待持有者登场。 |
| 文化作用 | 不同文明文物天然带来不同视觉符号，让角色有更强辨识度。 |
| 商业边界 | 可以参考文物形态和文化语义，但不要直接复刻博物馆照片、扫描图或展陈图，应做原创化梦域投影设计。 |

### 圣遗物 Rank

圣遗物可以有 Rank，但 Rank 代表“文物本身的神话权重”，不直接等于角色强度，避免后续角色因为 Rank 低就被玩家天然认为不值得期待。

| Rank | 含义 |
|-|-|
| B | 地区级旧梦遗物。 |
| A | 文明级圣遗物。 |
| S | 王朝级 / 神庙级圣遗物。 |
| EX | 旧猫王朝核心遗物，通常与主线剧情或顶级角色相关。 |

### 怎么继续完善

后续可以把角色结构定为：**猫咪本体 + 职业 + 参考文明 + 圣遗物**。玩家不是随机认识一只猫，而是先在猫眠所档案、加载页提示、剧情台词中听说某个圣遗物，再期待持有这件圣遗物的小猫登场。

| 角色 | 文明方向 | 适配圣遗物方向 |
|-|-|-|
| 塞班 | 凯尔特 / 中世纪不列颠 | 巴特西盾牌、萨顿胡头盔、滑铁卢角盔、越王勾践剑。 |
| 奈芙蒂斯 | 古埃及 | 纳尔迈调色板、图坦卡蒙黄金面具、罗塞塔石碑、胡尼弗亡灵书。 |
| 铃音 | 日本 / 神道 / 音律 | 铜铎、曾侯乙编钟、新罗金冠，也可以结合神乐铃方向做原创化。 |

亚瑟王圣剑、圣杯、猫猫大魔王王冠这类物品更适合归为“传说圣遗物”，不强行放进现实文物体系。现实文物负责文化厚度，传说圣遗物负责主线剧情和高燃节点。

## 日本圣遗物补充

在“圣遗物”体系中，日本方向可以继续扩展。铃音目前使用铃铛作为核心识别物，因此她的圣遗物不应只参考泛东方铃铛，而应更明确地绑定日本神话和神道仪式。

推荐将铃音的核心圣遗物设定为：**天岩户神乐铃**。它参考神乐铃 Kagura Suzu、天宇受卖命与天岩户神话。传说天宇受卖命在天岩户前起舞，使天照大神重新显现。游戏中可以解释为：铃声并不只是仪式音乐，而是一次古老的梦域唤光仪式；铃音持有的神乐铃，是这段神话在梦境中的猫族投影。
