# 影丸/逐影者帧动画设计方案

<title>影丸/逐影者帧动画设计方案</title>

<callout emoji="💡">
**设计结论：**影丸的动画核心不是“厚重战士”，而是“小体型高速刺客”。参考图里的猫勇士比例、圆脸、短腿低重心可以保留，但盾牌感要弱化，动作语言改成双月牙弯刀、金色残影、瞬间背刺和光斑诱敌。整体应做到：可爱轮廓不变，出手像影子一样快。
</callout>

# 一、动画总原则

| 维度 | 设计要求 |
|-|-|
| 体态 | 保持参考图的三头身小猫比例，站姿微蹲，头部稳定，四肢动作快但幅度短。冲刺和攻击时身体压低，尾巴与披风形成速度方向。 |
| 性格 | 待机看似无聊，眼睛一直在扫目标；出手前没有夸张吼叫，而是瞬间消失、瞬间命中。反差要体现“嘴上漫不经心，实际全场清场”。 |
| 色彩与特效 | 本体以象牙白毛色、深褐面罩和护具为主；技能特效统一使用金色弧光、半透明残影、短促火花和月牙形轨迹。不要把特效做成大面积火焰，否则会偏离刺客感。 |
| 节奏 | 推荐 12fps 制作关键帧，游戏内可插值或抽帧。普攻和位移可以有 1 帧命中停顿，让高速动作被玩家看清。 |
| 剪影 | 双弯刀在身体两侧形成“月牙耳翼”轮廓，跑动时刀身后拖；释放技能时可出现 2-3 个淡金残影，但不要遮住主角脸部。 |

# 二、动作包总览

| 动作 | 帧数建议 | 循环/一次性 | 重点表现 |
|-|-|-|-|
| 待机 Idle | 8 帧 | 循环 | 看似放松，眼神追踪；披风和尾巴轻动。 |
| 移动 Run | 8 帧 | 循环 | 低重心疾跑，脚下短尘，披风和刀向后拖影。 |
| 普攻：弧光斩 | 6 帧 | 一次性 | 双刀交叉快斩，形成 X 形金色弧光。 |
| 小技能一：掠影 | 10 帧 | 一次性 | 化为残影穿过敌人，在目标背后重现。 |
| 小技能二：光斑诱引 | 12 帧 | 一次性 | 投出跳动光斑，影丸隐身，光斑落地小爆。 |
| 大招：百影追猎 | 20 帧 | 一次性 | 多段背后切入，残影围猎，最后月环收束。 |
| 受击 Hit | 4 帧 | 一次性 | 轻微后撤，不做笨重硬直，保留刺客敏捷。 |
| 死亡/退场 Down | 8 帧 | 一次性 | 烟雾散去，只剩短暂金色影痕，避免过度惨烈。 |

# 三、逐帧设计

## 1. 待机 Idle：8 帧循环

影丸待机不能像普通站桩角色。它应维持“随时会冲出去”的低张力：身体基本不大动，但眼睛、耳朵、尾巴和披风持续给出警觉信息。循环节奏建议 0.8 秒一轮。

| 帧段 | 画面动作 |
|-|-|
| F1 | 双脚站稳，双刀垂在身侧，头略偏向敌人方向。 |
| F2-F3 | 耳朵轻动，眼睛向侧前方扫，披风边缘上扬一小段。 |
| F4 | 眨眼一帧，表情仍冷淡，形成“看似没在意”的反差。 |
| F5-F6 | 右爪轻压刀柄，身体重心向前半步，尾巴向反方向平衡。 |
| F7-F8 | 披风落回，金色微光在刀刃边缘闪一下，然后回到 F1。 |

## 2. 移动 Run：8 帧循环

移动参考图中第三排“贴地冲刺”的姿态，但影丸要更像追踪猎手：头低、眼睛锁定、四肢步频快。身体不要上下弹太高，否则会变成可爱跑步，削弱刺客压迫感。

| 帧段 | 画面动作 |
|-|-|
| F1-F2 | 前脚落地，身体贴近地面，双刀后摆，披风被拉成三角形。 |
| F3-F4 | 后脚蹬地，脚下出现短尘；身后留下 1 个浅金残影。 |
| F5-F6 | 身体前探，尾巴横向压低，刀刃闪出短月牙线。 |
| F7-F8 | 换步收回，残影淡出，准备回到 F1。 |

## 3. 普攻：弧光斩，6 帧一次性

普攻是影丸最快、最常见的识别动作。它的关键不是大幅挥刀，而是“短预备、快交叉、清晰命中”。每次命中要能让玩家看到弧光印记挂上敌人，所以第 4 帧建议做 1 帧命中停顿。

| 帧 | 动作描述 | 特效 |
|-|-|-|
| F1 | 身体下压，双刀收到身体两侧。 | 刀刃出现细金边。 |
| F2 | 前爪踏出，头部锁定目标。 | 身后出现小残影。 |
| F3 | 双刀交叉斩出，身体几乎被刀光带动。 | X 形金色弧光展开。 |
| F4 | 命中停顿，眼神不变。 | 敌人身上出现小月牙印记。 |
| F5 | 双刀甩到斜后方，披风跟随。 | 弧光拉长后变淡。 |
| F6 | 脚步回收，接回待机或下一次普攻。 | 地面一小点金色碎光消散。 |

## 4. 小技能一：掠影，10 帧一次性

掠影承担“位移、穿透、背后出现”三个信息，必须让玩家看懂影丸从哪里消失、穿过了谁、最终站到哪里。建议将本体透明化和残影轨迹分层制作，避免快速移动时看成瞬移 BUG。

| 帧段 | 动作描述 |
|-|-|
| F1-F2 | 影丸压低身体，双刀向后收，脚下出现半月形起跑光。 |
| F3 | 本体拉成金色残影，原地只留一层淡轮廓。 |
| F4-F6 | 残影穿过路径上的敌人，每穿过一个敌人产生短促斩线和弧光印记。 |
| F7 | 目标背后先出现一对发光眼睛和刀刃轮廓。 |
| F8-F9 | 本体重现，双刀反握收刀，目标背后爆出小型金色切痕。 |
| F10 | 披风落下，进入可接普攻或大招的姿态。 |

## 5. 小技能二：光斑诱引，12 帧一次性

光斑诱引是影丸的“狡猾感”来源。动作不能做成普通投掷炸弹，而要像逗猫光点反过来诱导敌人：影丸随手一弹，敌人被光斑吸住视线，影丸自己进入隐身位移。

| 帧段 | 动作描述 | 特效 |
|-|-|-|
| F1-F2 | 影丸从刀尖挑出一个小金点，动作轻巧，表情不变。 | 金点像萤火一样亮起。 |
| F3-F5 | 金点向前弹跳，影丸身体透明度降低。 | 光点留下断续轨迹。 |
| F6-F8 | 敌人视线被吸引；影丸只剩淡淡轮廓和眼睛余光。 | 敌人头顶出现短暂注视线。 |
| F9-F10 | 光斑落地，形成小范围金色圆环。 | 圆环扩散，边缘带减速碎光。 |
| F11-F12 | 影丸从隐身中滑向侧后方，准备接掠影。 | 本体淡入，地面光点熄灭。 |

## 6. 大招：百影追猎，20 帧一次性

大招要做出“场上所有被标记者都逃不掉”的压迫感。不要把 3 秒实际战斗完全逐帧画成复杂连击，建议用主角关键姿态加残影层完成：玩家看到的是多个背刺瞬间，而不是一段难读的乱闪。

| 帧段 | 动作描述 |
|-|-|
| F1-F3 | 影丸立定，场上弧光印记同时亮起，双刀交叉在胸前。 |
| F4-F5 | 本体消失，原地留下猫眼形金光和披风残片。 |
| F6-F13 | 多个残影依次从敌人背后切入。每个命中点只显示 1 个主姿态、1 条斩线、1 帧敌人受击停顿。 |
| F14-F16 | 残影围成半圆，形成“百影”效果；金色月牙轨迹在场地中交错。 |
| F17-F18 | 影丸回到主目标背后，双刀向外展开，所有印记同时爆亮。 |
| F19-F20 | 月环收束，影丸背对目标收刀，敌人延迟受击。 |

# 四、特效与图层拆分

| 图层 | 内容 | 制作备注 |
|-|-|-|
| Body | 头、身体、四肢、尾巴 | 头部尽量稳定，表情变化少，用眼神表达冷静。 |
| Cloak | 披风、面罩边缘 | 冲刺和转身时作为速度方向的主要读图线。 |
| Weapon | 双月牙弯刀 | 普攻帧里允许刀身轻微拉长，形成更清楚的月牙剪影。 |
| Afterimage | 金色残影、本体透明残留 | 残影最多 3 层，透明度从 45% 到 10% 递减。 |
| Slash FX | X 形斩线、月牙弧光、背刺切痕 | 线条要薄、亮、短促，不做大面积爆炸。 |
| Mark FX | 弧光印记、印记爆亮 | 敌人身上的标记建议做成小月牙，不要太大。 |
| Ground FX | 尘土、落地圆环、光斑减速圈 | 参考图中尘土可保留，但颜色降低饱和度，避免抢主角。 |

# 五、交付规格建议

- 每组动作导出透明 PNG 序列和一张 sprite sheet，保留同一画布尺寸，推荐 512×512 或 768×768，具体看游戏内角色显示尺寸。
- 所有动作 pivot 固定在脚底中心，掠影和大招不能因为残影外扩改变 pivot。
- 命名建议：Kagemaru_Idle_01、Kagemaru_Run_01、Kagemaru_AttackArc_01、Kagemaru_SkillShadowDash_01、Kagemaru_SkillLureSpot_01、Kagemaru_UltHundredHunt_01。
- 若资源预算有限，优先完成 Idle、Run、弧光斩、掠影、大招。光斑诱引可以先用 8 帧简化版，后续再补敌人注视细节。

# 六、给美术的重点提醒

<callout emoji="💡">
参考图的可爱猫勇士气质可以保留，但影丸不是举盾小骑士。请把盾牌防御感替换为双刀进攻感，把厚重冲撞替换为贴地滑行，把大爆炸替换为金色细弧光。玩家第一眼应读到“它很小、很快、很会绕后”。
</callout>
