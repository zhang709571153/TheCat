# 2026-06-02 · Hades 神明祝福与肉鸽随机系统拆解

<title>2026-06-02 · Hades 神明祝福与肉鸽随机系统拆解</title>

<callout emoji="📌">
**口径说明：**本文按《Hades》一代整理，不混入《Hades II》的神明祝福体系。核心结论是：Hades 的随机不是“随机给数值”，而是把攻击、特殊、投弹、冲刺、召唤、武器锤子、房间路线和稀有度做成可选择的组合模块。玩家每局都在“被随机影响”，但又总能通过门、信物、祝福槽位和搭配判断掌控方向。
</callout>

# 一、神明祝福系统怎么运作

Hades 的祝福是单局内生效的能力升级。祝福通常来自奥林匹斯神明，也可以从卡戎商店购买；玩家死亡回到冥府后，本局祝福会消失。它不是永久养成，而是每局 build 的主体。

| 祝福槽位 | 英文命名规律 | 设计作用 |
|-|-|-|
| Attack / 普攻 | Strike | 改变主要输出方式，例如上弱化、雷链、厄运、暴击。 |
| Special / 特殊技 | Flourish | 改变武器副动作，让同一武器出现不同战斗节奏。 |
| Cast / 投弹 | Shot / Flare | 把投弹变成短爆、追踪箭、刀阵、酒雾、冰晶炮台等。 |
| Dash / 冲刺 | Dash | 让位移同时产生防御、控场或输出价值。 |
| Call / 召唤 | Aid | 通过神力槽释放大招，提供爆发、无敌、魅惑、持续雷击等。 |

重要的是，核心槽位通常不能叠同类神明祝福。比如投弹不能同时既是阿佛洛狄忒的 Crush Shot，又是狄俄尼索斯的 Trippy Shot。系统会允许替换，并用更高稀有度补偿。这让玩家每次选择都不是“多拿一点”，而是“切换一条构筑路线”。

# 二、20 个代表性神明祝福

下面不是完整全量表，而是适合做设计参考的 20 个代表祝福。它们覆盖了伤害、控制、防御、召唤、场地、资源、机动和组合触发几类典型模式。

| 序号 | 神明 | 祝福 | 作用摘要 | 设计启发 |
|-|-|-|-|-|
| 1 | 宙斯 Zeus | Lightning Strike | 普攻命中后产生链状闪电。 | 适合高频普攻武器，低基础伤害也能通过跳电清群。 |
| 2 | 宙斯 Zeus | Thunder Dash | 冲刺时对附近敌人落雷。 | 把“躲避动作”转化成输出动作，提升走位收益。 |
| 3 | 宙斯 Zeus | Static Discharge | 雷电效果附加 Jolted，敌人攻击时受伤。 | 让单一元素有二段状态，形成“雷击→触发惩罚”的链路。 |
| 4 | 波塞冬 Poseidon | Tempest Strike | 普攻增强，并击退敌人。 | 同样是普攻强化，但手感从伤害变成空间控制。 |
| 5 | 波塞冬 Poseidon | Tidal Dash | 冲刺造成范围伤害并击退。 | 冲刺可以变成“推线工具”，适合防守地图借鉴。 |
| 6 | 波塞冬 Poseidon | Razor Shoals | 击退效果会施加 Rupture。 | 把控制动作转成持续伤害，让击退不只是位移。 |
| 7 | 雅典娜 Athena | Divine Dash | 冲刺造成伤害并反弹弹幕。 | 经典防御祝福，让新手用一个动作同时完成闪避和反制。 |
| 8 | 雅典娜 Athena | Divine Strike | 普攻增强并可反弹。 | 把主动攻击改成安全输出，适合承压角色。 |
| 9 | 雅典娜 Athena | Deathless Stand | 强化复活后的无敌时间，并立即补一层死亡抗拒。 | 失败保护也能成为祝福内容，降低局内崩盘感。 |
| 10 | 阿佛洛狄忒 Aphrodite | Heartbreak Strike | 普攻增伤并施加 Weak，降低敌人伤害。 | 高倍率伤害与减伤绑定，形成“高伤但偏防守”的神明风格。 |
| 11 | 阿佛洛狄忒 Aphrodite | Crush Shot | 投弹变成短距离宽范围爆发并施加 Weak。 | 改变技能射程和风险，让同一个 Cast 变成近身爆发。 |
| 12 | 阿瑞斯 Ares | Curse of Agony | 普攻施加 Doom，延迟爆发伤害。 | 延迟结算制造节奏差，适合“标记后爆炸”的状态设计。 |
| 13 | 阿瑞斯 Ares | Slicing Shot | 投弹变成向前移动的刀刃旋涡。 | Cast 不只是弹丸，可以变成移动场地伤害。 |
| 14 | 阿尔忒弥斯 Artemis | Deadly Strike | 普攻增强，并有概率暴击。 | 暴击路线天然适合高单次伤害或高频命中两类构筑。 |
| 15 | 阿尔忒弥斯 Artemis | Support Fire | 攻击、特殊技或投弹后发射追踪箭。 | 给所有动作附加小追踪弹，是“任何流派都能拼”的通用组合件。 |
| 16 | 狄俄尼索斯 Dionysus | Drunken Strike | 普攻施加 Hangover，造成持续中毒式伤害。 | 让高频攻击变成叠层玩法，适合持续输出和扩散设计。 |
| 17 | 狄俄尼索斯 Dionysus | Trippy Shot | 投弹生成 Festive Fog 酒雾区域。 | 投弹从命中技能变成区域控制，改变站位策略。 |
| 18 | 得墨忒耳 Demeter | Frost Strike | 普攻增强并施加 Chill，降低敌人速度。 | 减速元素最适合防守、风筝和聚怪。 |
| 19 | 得墨忒耳 Demeter | Crystal Beam | 投弹放置冰晶，持续向前发射光束。 | 把投弹变成临时炮台，形成位置经营。 |
| 20 | 赫尔墨斯 Hermes | Greatest Reflex | 增加连续冲刺次数。 | 不改变伤害，但直接改变玩家操作上限和生存手感。 |

<callout emoji="✅">
**对《梦境支配者》的直接启发：**每个状态标签最好有“主效果 + 二段联动”。例如缓速不只降速，还可以触发奈芙增伤；标记不只被看见，还可以让影丸大招追猎；护盾不只是减伤，还可以让塞班在床边产生击退波。这样肉鸽组合度才会来自技能关系，而不是单纯数值加成。
</callout>

# 三、除了祝福，还有哪些“技能/能力层”

Hades 好玩的原因之一，是它没有只靠祝福撑随机。玩家每局的战斗形态来自多层系统叠加：基础动作、武器、武器形态、祝福槽位、锤子改造、投弹、召唤、永久天赋和局内资源共同决定。

| 能力层 | 内容 | 设计作用 |
|-|-|-|
| 基础动作 | 普攻、特殊技、投弹、冲刺、召唤 | 祝福优先围绕这些动作插槽改造，玩家容易理解。 |
| 武器 | 剑、矛、盾、弓、拳套、炮 | 每种武器先给一套完全不同的操作骨架。 |
| 武器形态 | 同一武器的不同 Aspect | 改变基础动作的倾向，比如更适合暴击、投弹、冲刺或远程。 |
| Daedalus Hammer | 单局武器改造，最多通常拿两次 | 直接改变武器动作，有些会改变整局打法。 |
| Mirror of Night | 永久天赋，如复活、额外冲刺、投弹规则等 | 提供长期成长，并让玩家在开局前设定偏好。 |
| Keepsakes | 信物，可影响下一次神明祝福或给局内被动 | 给玩家有限控随机的手段，避免完全看脸。 |
| Companions | 后期解锁的召唤伙伴 | 提供额外爆发按钮，丰富 Boss 战节奏。 |
| Pact of Punishment | 通关后自选难度词条 | 让熟练玩家主动改变敌人、Boss、时间和资源压力。 |

这里最值得抄的不是系统数量，而是“层级分工”：武器负责基础手感，祝福负责局内方向，锤子负责动作变体，信物负责控随机，永久天赋负责长期成长。每一层都不抢另一层的职责。

# 四、每局随机元素有哪些

Hades 的随机元素可以分成四类：能力随机、路线随机、资源随机、事件随机。它们不是同时一股脑砸给玩家，而是以房间门选择的形式逐步出现，所以玩家每次都能做小决策。

| 随机元素 | 表现 | 为什么有效 |
|-|-|-|
| 神明祝福 | 每次遇到神明，从 3 个候选中选 1 个。 | 随机给方向，但由玩家做最终选择。 |
| 祝福稀有度和等级 | Common、Rare、Epic、Heroic；Pom 可升级已有祝福。 | 同一祝福也会因品质不同产生期待感。 |
| Duo / Legendary | 满足前置后出现组合或顶级祝福。 | 鼓励玩家主动规划神明组合，而不是乱拿。 |
| Chaos Boon | 先承受几房间惩罚，再获得强力加成。 | 提供风险换收益的赌博感。 |
| Daedalus Hammer | 随机给武器动作改造。 | 同一把武器每局手感都可能不同。 |
| 房间奖励 | 祝福、金币、血量、Pom、锤子、资源等。 | 门口先显示奖励符号，玩家可以选路线。 |
| 卡戎商店和 Well | 用金币买祝福、血量、Pom 或临时效果。 | 把资源管理加入随机路线。 |
| NPC 房间 | 西西弗斯、欧律狄刻、普特洛克勒斯等给特殊选择。 | 在高压战斗中插入缓冲和故事奖励。 |
| Trials of the Gods | 二选一神明，没选的神会暂时惩罚你。 | 把选择做成戏剧冲突，同时给双奖励。 |
| 敌人与房间组合 | 不同房间形状、敌人组合、精英奖励。 | 让同一构筑也要面对不同操作题。 |
| Boss 变体 | 部分 Boss 有不同阶段或强化变化。 | 防止固定背板过快失效。 |
| 资源与长期成长 | 钥匙、黑暗、宝石、蜜露、泰坦之血等。 | 死亡后仍有推进，减少失败挫败感。 |

# 五、它好玩的核心原因

<callout emoji="⭐">
**一句话：**Hades 的爽点不是随机多，而是“随机给题，玩家解题”。每局有不确定性，但玩家始终能通过选门、选祝福、控神明、配武器、换路线来表达自己的判断。
</callout>

## 1. 祝福是“动词”，不是纯数值

很多肉鸽随机奖励只写“攻击 +10%”，玩家感受到的是数字变大。Hades 的祝福更像动词：反弹、击退、魅惑、厄运、暴击、酒雾、冰冻、链雷、刀阵、炮台。玩家拿到祝福后，动作的意义会变。例如冲刺原本是逃命，拿到 Tidal Dash 后变成推怪；拿到 Divine Dash 后变成弹幕反制；拿到 Thunder Dash 后变成输出。动作意义改变，手感才会变。

## 2. 神明有明确“元素性格”

每个神明都不是散乱技能包。宙斯是连锁雷电，波塞冬是击退和资源，雅典娜是反弹和防御，阿佛洛狄忒是 Weak 和高伤，阿瑞斯是 Doom 与刀阵，阿尔忒弥斯是暴击，狄俄尼索斯是 Hangover 和酒雾，得墨忒耳是 Chill，赫尔墨斯是速度与操作上限。玩家看到神明头像时已经能预判这局可能走向，这种识别度会降低学习成本。

## 3. 随机被“有限控制”包住

玩家不是完全被系统发牌。房间门会显示奖励类型，信物能提高指定神明出现概率，Fated Authority 可以重掷部分房间奖励，商店能用金币补关键资源，Duo 和 Legendary 有明确前置。随机带来变化，控制手段带来策略感。两者叠在一起，玩家会觉得“这局是我打出来的”，而不是系统随机送输赢。

## 4. 构筑来自跨层组合

一个好 build 往往不是单个祝福强，而是武器高频/低频、祝福状态、锤子改造、投弹机制和冲刺行为彼此咬合。高频拳套适合叠 Hangover 或触发链雷；高单次伤害武器适合暴击；防守和推线场景适合击退、反弹、减速。每一层都在问玩家：你这局到底想把哪个动作变成核心？

## 5. 死亡也在推进叙事和成长

Hades 把死亡做成叙事回流。失败后回家不是清零，而是见角色、送礼、解锁剧情、升级镜子、换武器、换信物。肉鸽的失败循环因此不只是惩罚，而是“又回家拿到新信息”。这点对《梦境支配者》很重要：如果猫猫守梦失败，也可以让猫眠所档案、角色台词、梦域情报和圣遗物线索继续推进。

# 六、给《梦境支配者》的转化建议

如果借鉴 Hades，建议不要先做大而全的道具池，而是先做一套小而清晰的“猫咪祝福/梦境强化”结构。P0 可以从 3 只初始猫各 3 个强化开始，再加 3 个梦境事件，保证每局至少能形成一个可感知流派。

| Hades 机制 | 梦境支配者对应做法 | P0 建议 |
|-|-|-|
| 神明祝福 | 猫眠所强化 / 圣遗物投影 / 梦域祝福 | 先做 9 个：塞班、奈芙、铃音各 3 个。 |
| Duo Boon | 双猫联携强化 | 先不做大池，只做 2-3 个验证组合感。 |
| Daedalus Hammer | 技能形态改造 | 每只猫至少 1 个“改变手感”的强化，而不是只加伤害。 |
| Chaos Boon | 梦魇交易 / 噩梦契约 | 先做 1 个风险收益事件，避免系统过重。 |
| Pom of Power | 强化已有技能等级 | P0 可简化为“升级当前已有强化”。 |
| Room Reward | 波次结束三选一 | 每波结束给 3 选 1，玩家逐步成型。 |

<callout emoji="✅">
**建议优先落地：**先做“塞班床边防线 / 奈芙炮台变体 / 铃音安眠铃变体”三组强化。因为它们能直接改变站位、防守、清怪和救场节奏，比单纯攻击力 +10% 更容易让玩家感到每局不同。
</callout>
