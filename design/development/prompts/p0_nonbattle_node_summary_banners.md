# P0 Non-Battle Node Summary Banners

## Scope

Produce three transparent symbolic UI banners for the route map current-node card:

- `thecat_ui_node_shop_summary_banner_512x160_v001`
- `thecat_ui_node_dreamevent_summary_banner_512x160_v001`
- `thecat_ui_node_restnest_summary_banner_512x160_v001`

## Style

- Hand-painted dreamglass UI language, matching the existing P0 route card frames.
- Transparent background, 512x160, Unity Sprite import.
- No typography; the banners should remain localization-safe.
- Symbolic non-cat UI only. Do not use cat bodies, cat face markings, starter-cat costumes, or derivatives of starter-cat turnarounds.

## Motifs

- Shop: fish-gold supply bag, coins, shelf line, cyan support glow.
- Dream event: moon cloud, split risk/reward arrows, purple and cyan motes.
- Rest nest: crescent nest, sleep wave, recovery plus, green-blue rest glow.
