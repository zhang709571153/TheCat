# P0 Route Node Icons Prompt Notes

## Scope

Batch 06 produces deterministic route-map icons for the non-Boss node types in
the 10-layer P0 route:

- defense
- elite
- partner
- shop
- dream event
- blessing offering
- rest nest

The Boss route icon remains source-derived from the locked Call Tyrant concept.

## Visual Direction

- Match the existing P0 HUD/status icon language: soft dream glow, high-contrast
  silhouette, readable at 32px and 64px.
- Use transparent 128x128 PNG sprites with a circular dream badge base.
- Keep icons symbolic rather than illustrative.
- Do not include cat bodies, cat faces, or starter-cat costume motifs in this
  batch.
- Do not reference or modify starter cat turnaround sources.

## Icon Reads

| Node Type | Read |
| --- | --- |
| Defense | bed shield, stable blue defense |
| Elite | warning diamond, red-purple pressure |
| Partner | linked dream circles, safe recruit |
| Shop | fish treat economy |
| Dream Event | cloud and spark choice |
| Blessing Offering | altar/crown authority choice |
| Rest Nest | crescent nest recovery |

## Builder

Generated by:

`design/development/tools/build_route_node_icons.ps1`

The builder uses deterministic vector drawing through `System.Drawing` and
writes Unity Sprite `.meta` files with `TheCatP0ImportSettings:v1`.
