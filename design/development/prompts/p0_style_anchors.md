# P0 Style Anchor Prompts

Status: draft only. Do not generate until Unity code validation is back to a
clean state and the batch can be reviewed against the manifest.

## Shared Negative Constraints

- no humanoid cat girls
- no photorealism
- no glossy generic mobile RPG render
- no direct museum replica or copyrighted display photo
- no gore
- no text baked into image unless specifically requested for logo work

## Bedroom Dream Anchor

Hand-painted dreamlike bedroom battlefield, cozy bed as the protected objective,
soft blue lavender sleep light, subtle dream fog, readable top-down / angled
gameplay composition, space for UI, cat-scale props, litter box and feeder
silhouettes, painterly soft edges, gentle but tense night mood.

## Starter Cats Lineup Anchor

Three non-humanoid stylized cats in a hand-painted dream game style, compact
animal bodies, readable silhouettes: Saiban heavy defender with silver oath
shield and sun-gold accents, Nephthys elegant controller with Egyptian obelisk
and moon-sand symbols, Suzune small healer with kagura bell and torii sleep
circle, consistent palette, no human bodies, no costumes that hide cat anatomy.

## Black Mud Nightmare Anchor

Small crawling black-violet dream mud monster, soft sludge body, hostile red
core crack, simple readable silhouette for gameplay, nightmare intrusion from
sleep fog, hand-painted, not gory, not realistic horror anatomy.

## P0 Status Icon Anchor

Five small readable game status icons on transparent background, unified
painterly UI style: sleep stable as blue crescent note, slow as moon-sand spiral,
knockback as silver impact burst, mark as royal eye sigil, shield as silver-blue
oath shield rim. Clean silhouettes at 64x64.
